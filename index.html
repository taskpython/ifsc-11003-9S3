    • Thoughts on Progressive Enhancement

    In my own words, why I think progressive enhancement is good for web development is because it helps with the user experience, making the website easily accessible to a wide variety of devices. It starts with a default website design that allows other CSS and JS files to accommodate different devices with different modularity, allowing the site to deliver core content no matter a CSS or JS failure. Its important because not all devices are made the same.
